Ebay Exercise
=============

Clone project
-------------

```bash
git clone https://github.com/GaryAghedo/ebay-exercise.git
cd ebay-exercise
```

How to run Program
----------
- single line command: `sbt run` e.g.
```bash
$ sbt run
```

How to run Test
----------
- single line command: `sbt test` e.g.
```bash
$ sbt test
```
